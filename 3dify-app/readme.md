Moderne Smartphones bieten die Möglichkeit, Fotos in einem speziellen Modus zu schießen, um sie danach mit einem Bokeh-Effekt auszustatten. Im Vortrag erfahrt ihr, wie dieser Effekt auf aktuellen iPhones mit Multi-Kamera-Arrays über Methoden aus dem Bereich Computer Vision funktioniert. Außerdem wird diskutiert, wie Machine Learning auf dem Smartphone eingesetzt werden kann, um diesen Effekt auch auf Systemen mit nur einer Kamera zu simulieren. Auf Basis dieser Algorithmen können auch andere interessante Effekte erstellt werden. Im Vortrag wird die 3D-ify App demonstriert, welche diese Verfahren implementiert, um Fotos eines iPhones einen interessanten 3D-Parallax-Effekt zu verleihen.
